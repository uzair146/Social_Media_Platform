# 🚀 Social Network Management System

### *Data Structures Lab Project | C++ Console Application*

A fully functional console-based social networking platform developed using core **Data Structures & Algorithms** concepts in **C++**.
This project demonstrates how real-world applications can be built using fundamental DSA concepts like Hash Tables, Graphs, AVL Trees, Queues, Stacks, and Linked Lists.

---

# ✨ Project Overview

The **Social Network Management System** is a mini social media platform where users can:

* Create accounts
* Add friends
* Create posts
* Like posts
* Send messages
* View stories
* Receive notifications
* Analyze user activity

The main goal of this project was to implement complex **Data Structures** in a practical and interactive way.

---

# 🔥 Key Features

## 👤 User Management

* User Registration
* Secure Login System
* Logout Functionality
* Search Users
* Delete Account
* Display All Users

---

## 🤝 Social Network Graph

* Add Friends
* Remove Friends
* View Friend List
* BFS Traversal
* DFS Traversal

Graph implementation simulates a real social network connection system.

---

## 📝 Posts & Feed

* Create Posts
* Delete Posts
* View Personal Posts
* Like Posts
* Interactive Feed Navigation

Feed navigation is implemented using a **Doubly Linked List**.

---

## 📸 Stories System

* Add Stories
* Circular Story Viewing System

Stories are implemented using a **Circular Linked List**.

---

## 🔔 Notification System

* Like Notifications
* Message Notifications
* Process Notifications Queue

Implemented using **Queue Data Structure**.

---

## 💬 Messaging System

* Send Messages
* View Latest Message
* Display Conversations
* Delete Messages

Chat system is implemented using **Stacks**.

---

## 📊 Analytics System

* Most Active Users
* Most Liked Posts
* Sorted Analytics
* Range Queries

Implemented using **AVL Trees** for optimized performance.

---

# 🧠 Data Structures Used

| Data Structure       | Purpose                    |
| -------------------- | -------------------------- |
| Hash Table           | User Storage & Fast Lookup |
| Linked List          | User & Post Management     |
| Doubly Linked List   | Feed Navigation            |
| Circular Linked List | Stories System             |
| Queue                | Notifications              |
| Stack                | Messaging                  |
| Graph                | Friend Network             |
| BFS / DFS            | Social Traversal           |
| AVL Tree             | Analytics & Ranking        |
